# Quickstart

By the end of this page you'll have created multiple pages, added navigation, fetched data, and styled your app.

## Prerequisites

- A Next.js project (see [Installation](/documentation/installation))
- Your dev server running on `http://localhost:3000`

## 1. Create your first page

Every file inside `app/` that exports a default React component becomes a route.

<Steps>
  <Step>
    ### Edit the home page

    Open `app/page.tsx` and replace its content:

    ```tsx title="app/page.tsx"
    export default function Home() {
      return (
        <main>
          <h1>Welcome to my app</h1>
          <p>Built with Next.js 15</p>
        </main>
      )
    }
    ```

    Save the file. The browser auto-refreshes with your changes.
  </Step>
  <Step>
    ### Add a second page

    Create a new file at `app/about/page.tsx`:

    ```tsx title="app/about/page.tsx"
    export default function About() {
      return (
        <main>
          <h1>About</h1>
          <p>This is the about page.</p>
        </main>
      )
    }
    ```

    Navigate to [http://localhost:3000/about](http://localhost:3000/about) to see it.

    <Callout type="info">
      The folder name becomes the URL segment. `app/about/page.tsx` → `/about`.
    </Callout>
  </Step>
  <Step>
    ### Add navigation

    Update your layout to include links between pages:

    ```tsx title="app/layout.tsx"
    import Link from 'next/link'

    export default function RootLayout({
      children,
    }: {
      children: React.ReactNode
    }) {
      return (
        <html lang="en">
          <body>
            <nav>
              <Link href="/">Home</Link>
              <Link href="/about">About</Link>
            </nav>
            {children}
          </body>
        </html>
      )
    }
    ```

    <Callout type="warn">
      Always use `next/link` instead of `<a>` tags for internal navigation. It enables client-side transitions and prefetching.
    </Callout>
  </Step>
</Steps>

## 2. Fetch data

Server Components can be `async` — you can `await` data directly in your component.

```tsx title="app/posts/page.tsx"
interface Post {
  id: number
  title: string
  body: string
}

export default async function Posts() {
  const res = await fetch('https://jsonplaceholder.typicode.com/posts?_limit=5')
  const posts: Post[] = await res.json()

  return (
    <main>
      <h1>Posts</h1>
      <ul>
        {posts.map((post) => (
          <li key={post.id}>
            <h2>{post.title}</h2>
            <p>{post.body}</p>
          </li>
        ))}
      </ul>
    </main>
  )
}
```

<Callout type="info">
  This `fetch` runs on the **server** at request time. The data is already rendered in the HTML sent to the browser — no loading spinner needed.
</Callout>

## 3. Add styling

Next.js supports multiple styling approaches out of the box:

<Tabs items={["CSS Modules", "Tailwind CSS", "Global CSS"]}>
  <Tab value="CSS Modules">
    ```css title="app/page.module.css"
    .hero {
      padding: 4rem 2rem;
      text-align: center;
    }

    .title {
      font-size: 2.5rem;
      font-weight: 700;
    }
    ```

    ```tsx title="app/page.tsx"
    import styles from './page.module.css'

    export default function Home() {
      return (
        <main className={styles.hero}>
          <h1 className={styles.title}>Welcome</h1>
        </main>
      )
    }
    ```
  </Tab>
  <Tab value="Tailwind CSS">
    If you selected Tailwind during `create-next-app`, it's already configured:

    ```tsx title="app/page.tsx"
    export default function Home() {
      return (
        <main className="flex min-h-screen flex-col items-center justify-center p-24">
          <h1 className="text-5xl font-bold">Welcome</h1>
        </main>
      )
    }
    ```
  </Tab>
  <Tab value="Global CSS">
    Import your global stylesheet in the root layout:

    ```css title="app/globals.css"
    * {
      box-sizing: border-box;
      margin: 0;
      padding: 0;
    }

    body {
      font-family: system-ui, -apple-system, sans-serif;
      line-height: 1.6;
    }
    ```

    ```tsx title="app/layout.tsx"
    import './globals.css'

    export default function RootLayout({ children }) {
      return (
        <html lang="en">
          <body>{children}</body>
        </html>
      )
    }
    ```
  </Tab>
</Tabs>
